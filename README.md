# Graph-Based Person Management Project

A C++ term project built with CMake.  
The project is organized around core modules such as `Graph`, `Person`, and `utils`, and appears to model people and their relationships using graph-based logic.

## Project Structure

```text
.
├── CMakeLists.txt
├── main.cpp
├── Graph.cpp
├── Graph.h
├── Person.cpp
├── Person.h
├── utils.cpp
├── utils.h
└── README.md
```

## Technologies

* C++
* CMake
* Object-Oriented Programming
* Graph-based data modeling

## Features

* Person-based data modeling
* Graph structure and relationship handling
* Modular C++ design with header/source separation
* Build configuration with CMake

## How to Build

```bash
mkdir build
cd build
cmake ..
cmake --build .
```

## How to Run

After building, run the generated executable from the `build` folder.
